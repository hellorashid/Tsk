# tsk.lol

A cozy task management and scheduling app that helps you organize your life. 


## Main Features

### Task Management
- **Create and organize tasks** with descriptions and subtasks
- **Folders** to group related tasks together
- **Track when you complete things** with activity history
- **Keyboard shortcuts** for power users who like to move fast

### Scheduling
- **Timeline and Agenda views** to see your day at a glance
- **Schedule tasks** to specific times (or let it find the next free slot)
- **Create events** and keep track of your calendar
- **Weather integration** that shows sunrise, sunset, and daily forecasts
- **Focus session tracking** so you know how long you actually worked on things

### Focus Mode
- **Full-screen focus view** to block out distractions
- **Built-in stopwatch timer** to see how long you've been working
- **Edit tasks** right there in focus mode
- **Mark tasks as done** directly from the timer

### Customization
- **Customizable themes** with accent colors and dark/light modes, with more coming soon!
- **Open source** - fork, edit, and make it yours

### User Experience
- **Dynamic Island** (desktop) - a slick expandable interface for creating tasks and events
- **Command palette** - hit `/` to access quick actions
- **Mobile-optimized** with a responsive design that actually works on phones
- **Progressive Web App** - install it and use it offline


### Data Storage
Everything is stored using Basic.tech, so your data stays private and syncs across your devices when you're signed in.

### Keyboard Shortcuts
- `t` - Create new task
- `e` - Create new event
- `/` - Open command palette
- `Tab` - Cycle through folders
- `Arrow Left/Right` - Navigate tasks
- `Arrow Up/Down` - Navigate schedule events
- `Space` - Enter focus mode (when task selected)
- `Escape` - Close modals/drawers

## Getting Started

1. Head to [tsk.lol](https://tsk.lol) or run it locally (see [CONTRIBUTING.md](./CONTRIBUTING.md))
2. Start typing in the input field to create tasks
3. Organize things into folders using the folder bar
4. Schedule tasks by selecting them and adding them to your calendar
5. Hit focus mode when you need to actually get stuff done

## Tech Stack

- React + TypeScript
- Vite
- Basic.tech (database + auth + sync)
- Tailwind CSS
- Framer Motion
- Open-Meteo API (weather)
